# Olist — Predictive Analysis & Customer Segmentation

## Objective

Build an end-to-end e-commerce analytics project on the Olist dataset:

- Customer-level feature engineering (RFM + delivery + satisfaction)
- Customer segmentation (clustering)
- Predictive modeling (e.g., revenue / purchase frequency by segment)
- Business insights and recommendations

## Project structure

- notebooks/ : step-by-step notebooks (00 → 06)
- src/ : reusable Python functions
- data/ : local datasets (ignored by git)
- reports/ : figures and outputs

## How to run (local)

1) Create and activate `.venv`
2) Install dependencies: `pip install -r requirements.txt`
3) Run notebooks in `notebooks/`

> Dataset (Olist) must be downloaded locally and placed in `data/raw/`.
